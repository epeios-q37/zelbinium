# *iSH*

[*iSH*](https://ish.app/) est un *shell* *Linux* pour *iOS* qui permet d'installer un interpréteur *Python*.  On peut ainsi directement exécuter ses programmes *Python* directement sur son smartphone ou sa tablette *Apple* sans avoir à utiliser *Replit*.
