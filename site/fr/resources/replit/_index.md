# *Replit*

*NOTA* : Anciennement, *Replit* était nommé *Repl.it* et avait pour adresse https://repl.it. On trouvera donc donc ces anciennes appellations et adresse dans les documentations qui n'ont pas été mises à jour.

*Replit* est un [environnement de développement](https://fr.wikipedia.org/wiki/Environnement_de_d%C3%A9veloppement) en ligne, auquel on peut accéder avec un navigateur web, y compris à partir d'un smartphone ou d'une tablette. Outre un certains nombre d'outils qui facilitent le développement de programmes dans de nombreux langage, *Replit* permet notamment également d'exécution ces programmes sans avoir rien à installer, comme on peut le constater avec les nombreux encarts présents sur ce site et qui s'appuient sur *Replit*.

Dans *Replit*, un programme est stocké dans ce qui est appelé un *repl*. Un *repl* peut contenir plusieurs 
programmes, et, pour un programme donné, son *repl* contient l'ensemble des fichiers qui le constituent. 

On peut exécuter les programmes d'un *repl*, pour peu que son propriétaire l'ai rendu publique, sans posséder de compte *Replit*, mais pour modifier un *repl*, ou pour créer ses propres *repl*, un compte *Replit* est nécessaire. 

*Replit* propose des fonctionnalités payantes, mais la création d'un compte ainsi que les fonctions de base, largement suffisantes dans le cadre de *Zelbinium*, sont gratuites.

*Replit* est accessible à l'adresse <https://replit.com>.
