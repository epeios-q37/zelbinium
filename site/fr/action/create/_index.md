# Créer un programme

Créer un programme dans *Replit* signifie crée un ou plusieurs fichiers contenant le code source du programme. Sous *Replit*, les programmes sont stockés dans des *repl*. Un *repl* peut contenir plusieurs programmes, mais il est généralement préférable de créer un *repl* par programme.

Pour ce faire, on se rend à l'adresse https://replit.com, et on se connecte à son compte si ce n'est déjà fait. On clique alors sur le bouton *Create Repl*, ce qui provoque l'apparition d'un *popup*.

![](./OnCreateRepl.png)

Dans ce *popup*, on sélectionne dans le champ *Template* le langage avec lequel on désire programmer, ici *Python*, on change éventuellement le nom du *repl* dans le champ *Title*, puis on clique sur le bouton *Create Repl*.

![](./AfterCreateRepl.png)

On se retrouve alors dans l'environnement de développement, et l'on peut commencer à taper les lignes de code du nouveau programme et à lancer leur exécution en cliquant sur le bouton *Run*.

Il y a sur ce site de nombreux programmes pouvant servir d'inspiration pour créer ses propres programmes.
