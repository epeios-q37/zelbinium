# Explorer le code source d'un programme

Le [code source](https://fr.wikipedia.org/wiki/Code_source) d'un programme est la liste des instructions que l'ordinateur doit exécuter. Ce code source peut être découpé en plusieurs fichiers.

<iframe src="https://replit.com/@Zelbinium/Messages?embed=true" width="500" height="360"></iframe>

En cliquant sur *Show files*, une liste de fichiers va s'afficher.

![](./ExploreFileList.png)

Le fichier qui nous intéresse est *main.py*. Les autres fichiers sont des fichiers qui sont gérés par l'environnement d'exécution du programme et dont on n'a pas à s'occuper dans l'immédiat.

Pour voir le contenu du fichier *main.py*, il suffit de cliquer sur son nom. 

![](./ExploreFileView.png)

Il est possible que la liste des fichiers reste affichée, réduisant la zone d'autant affichage du fichier. Le bouton *Show files* est alors renommé en *Hide files*. Cliquer dessus cache le panneau affichant la liste des fichiers, le fichier affiché occupant alors toute la largeur de l'encart.

En cliquant sur le bouton *Open on Replit* situé en-haut à droite de l'encart, ou encore sur le lien <a href="https://replit.com/@Zelbinium/Messages" target="_blank">https<span/>://replit.com/@Zelbinium/Messages</a>, un nouvel onglet va s'ouvrir dans lequel se trouvera un bouton nommé *Show code* dont la fonctionnalité est la même que le bouton *Show files* mentionné ci-dessus.

À la [page suivante](../modify/), on verra comment modifier le code source du programme.
